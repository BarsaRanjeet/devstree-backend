## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
```required
## To run app you must have nestjs installed in your system

# step 1
$ npm i -g @nestjs/cli

# step 2
$ npm install

# add credentials in env file
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
